# PawPrints Veterinary Clinic Template

Welcome to the **PawPrints Veterinary** HTML template! This is a modern, fully responsive, and beautifully designed web template specifically crafted for veterinary clinics, animal hospitals, and pet care centers.

## 🚀 Features

- **Modern & Premium Design**: Clean, glassmorphism elements, beautiful micro-interactions, and professional aesthetics.
- **Dark Mode & RTL Support**: Built-in support for Dark Mode and Right-to-Left (RTL) languages for global accessibility.
- **Fully Responsive**: Flawless experience across desktops, tablets, and mobile devices.
- **Dashboard Layouts**: Includes dedicated layouts for Client Portal and Admin Dashboard.
- **Vanilla Technologies**: Built using pure HTML, CSS, and Vanilla JavaScript. No heavy frameworks!

## 🛠️ Usage & Customization

1. **Viewing the Template**: Simply open the root `index.html` file in any modern web browser to view the template index, which links to all available pages.
2. **Global Variables**: To change the primary theme colors, typography, or spacing, simply edit the CSS Custom Properties (`:root`) located at the very top of `assets/css/style.css`.
3. **Dark Mode**: Dark mode colors are handled via the `[data-theme="dark"]` selector in the CSS files.
4. **Icons**: This template utilizes **Phosphor Icons**. You can easily change icons by updating the `ph-*` class names in the HTML.

## 📱 Browser Support

Supports all modern browsers including Chrome, Safari, Firefox, and Edge.
